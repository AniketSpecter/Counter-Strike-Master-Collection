# Public Roadmap

This roadmap describes intended directions, not guaranteed delivery dates.

## Delivered in v1.0.5

- Counter-Strike WW2 added with an in-launcher game-mode and graphics picker.
- Half-Life added with a direct installer download flow.
- Steam scan now discovers all games in `steamapps\common` automatically.
- Thumbnail images for the Half-Life family and HellStrike are now auto-matched.
- Three-dot context menu positioning fixed; right-click on cards supported.

## Stability and downloads

- Continue improving provider compatibility, retry behavior, and actionable error messages.
- Expand download diagnostics while keeping personal paths and credentials private.
- Improve archive integrity metadata and repair verification.
- Add clearer disk-space and permission checks before large downloads.

## Library experience

- Improve existing-game and Valve-mod discovery.
- Expand move, backup, restore, and library-management safeguards.
- Refine game-specific launch presets while preserving user-selected executables.
- Continue improving playtime history and installed-game presentation.

## Community and updates

- Publish launcher updates, patch notes, and maintenance notices through GitHub Releases.
- Use Issues for reproducible bugs and feature requests.
- Use Discussions and Discord for community support and collection feedback.
- Add more public screenshots and user documentation as features mature.

Vote on existing feature requests with reactions instead of creating duplicates.
